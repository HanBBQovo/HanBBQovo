### About

全栈开发，偏 **Web 产品** 与 **后台系统**。

日常主要写 **Vue / React + TypeScript** 做前端，用 **Go / Node** 做服务端；也会用 Rust、Python 做一些工具和小服务。喜欢把复杂流程做成干净好用的界面和稳定可维护的后端。

目前在做：管理后台、自动化工具、以及一些开源小项目。
